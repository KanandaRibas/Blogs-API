## Blogs-Api

:large_blue_diamond: Neste projeto desenvolvi uma API e um banco de dados para a produção de conteúdo para um blog!  :mega:

:large_blue_diamond: Tecnologias utilizadas:
- [Node.js](https://nodejs.org/en/about)
- [Sequelize](https://sequelize.org/)
- [MySQL](https://www.mysql.com/)
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [Docker](https://docs.docker.com/get-started/)

</br>

:large_blue_diamond: Desenvolvi endpoints para fazer um CRUD - `Create (criar), Read (ler), Update (atualizar) e Delete (apagar)`, conectando ao banco de dados seguindo os princípios do REST;

:large_blue_diamond: Para fazer um post é necessário usuário e login, gerando um token para autenticação; 

:large_blue_diamond: Para manipular as informações do banco de dados foi trabalhado também, a relação entre as tabelas `categories`, `posts` e `users`.

:large_blue_diamond: O projeto foi desenvolvido durante o curso de desenvolvimento Web Full Stack da [Trybe](https://app.betrybe.com).

  - As configurações iniciais do sequelize, e os arquivos da pasta /seeders foram fornecidas pela Trybe.
  - Desenvolvi os arquivos dentro de /src, separando por responsabilidade em camadas Controllers, Services, Models e Migrations.
